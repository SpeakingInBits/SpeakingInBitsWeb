# SpeakingInBits Web
This project is being made to host courses and lecture material for students. Currently there will be one
instructor managing the courses. Students will eventually be able to track their progress through each course,
download lecture resources, and download assignments/projects to complete.

## Requirements
This project is being built with:
- [Visual Studio 2022](https://visualstudio.microsoft.com/vs/preview/) and .NET 6 (which is included)
  - Install the ASP.NET and Web Development Workload
